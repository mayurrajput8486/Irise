# 🚀 Irise - Frontend Learning Repository

Welcome to the **Irise** repository 👨‍💻
This repository is designed to help students and developers learn **Frontend Development** from basics to advanced concepts.

---

## 📌 What You Will Learn

This repository contains practical examples and code for:

* ✅ HTML (Basics to Advanced)
* ✅ CSS (Layouts, Flexbox, Grid, Styling)
* ✅ JavaScript (Core + DOM + ES6+)
* ✅ React.js (Components, Hooks, Projects)

---

## 📁 Folder Structure

```
Irise/
│
├── HTML/        → HTML practice files (audio, video, iframe, forms, etc.)
├── CSS/         → Styling concepts and layouts
├── JS/          → JavaScript concepts and examples
├── React/       → React projects and concepts
└── README.md
```

---

## 🎯 Purpose of This Repository

* Help students understand concepts with **real code examples**
* Provide **interview-ready knowledge**
* Practice **hands-on development**
* Build strong **frontend fundamentals**

## 🚀 How to Use This Repository

1. Clone the repository:

   ```bash
   git clone https://github.com/mayurrajput8486/Irise.git
   ```

2. Open in VS Code

3. Run HTML files using Live Server

4. Explore folders step by step
